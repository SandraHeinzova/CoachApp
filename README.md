# CoachApp – Moderní digitální zázemí pro sportovní kluby

CoachApp je webová platforma navržená pro efektivní správu a digitalizaci sportovních klubů. Projekt vznikl jako ročníková práce na FEI UPCE a zaměřuje se na zjednodušení organizace vedení sportovních týmů trenérům.

## 🚀 Živé demo a Landing Page
Aktuální stav vývoje a vizi produktu můžete sledovat zde:
**[CoachApp Landing Page](https://coachapp-landing.vercel.app)** 

## ✨ Klíčové vlastnosti (Vize produktu)
- **Organizace bez papírů:** Digitální správa soupisek a tréninkových plánů.
- **Taktická příprava:** Přehledné rozhraní pro trenéry i hráče.
- **Bezpečnost dat:** Moderní přístup k ochraně informací v klubu.
- **Vše na jednom místě:** Od docházky až po týmovou nástěnku.

## 🛠️ Použité technologie
- **Frontend:** React, HTML5, CSS3 (s důrazem na responzivní design)
- **Backend:** Java, Spring Boot, Spring Security
- **Vývojové prostředí:** Vite
- **Deployment:** Vercel

## 📁 Struktura projektu
- `/frontend` – Zdrojové kódy klientské části
- `/backend` – Návrh serverové části
- `/Database` – Schémata a návrh databáze
- `/Dokumentace` – Kompletní uživatelské a administrátorské příručky + produktový list
